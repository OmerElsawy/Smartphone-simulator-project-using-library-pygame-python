# 📱 Virtual Phone Simulator

A smart phone interface simulator built with **Python** and **Pygame**.  
This project provides an interactive mobile-like environment where users can launch applications, switch between them, listen to music, and play games such as Snake 🐍 within a single unified interface.

---

## ✨ Features

- 🔒 **Lock Screen** with password protection.
- 🕒 **Live Clock** displayed on the interface.
- 📱 **Built-in Applications**:
  - 🎮 Snake Game with animated head and body sprites.
  - 🧮 Calculator.
  - 🎵 Music Player with support for 3 audio tracks.
  - 💬 Messaging App connected to a Telegram Bot.
- 🎨 Elegant user interface inspired by modern smartphones.

---

## 🚀 Getting Started

### 1. Requirements

Make sure you have **Python 3.11+** installed.

### 2. Install Dependencies

```bash
pip install pygame pyTelegramBotAPI
```

### 3. Run the Project

```bash
python main.py
```
